# LP Factory — WebPanel (Streamlit)

Painel simples em Streamlit que se conecta ao **Engine FastAPI** do LP Factory via HTTP (X-API-KEY).
Este painel **não acessa Supabase direto** — tudo passa pelo Engine.

## Rodar local (dev)

### Requisitos
- Python 3.11+ (recomendado)
- Pip

1) Entre na pasta do painel:

cd webpanel_streamlit
