# Fluxo Simples de GitHub (Projeto Webpanel)

## Ferramenta recomendada
- GitHub Desktop (interface visual, sem comandos)

## Fluxo diário de trabalho

1. Abrir o projeto normalmente (VS Code / Antigravity / Cursor)
2. Fazer alterações no código
3. Abrir o GitHub Desktop
4. Conferir os arquivos alterados
5. Escrever uma mensagem curta de commit
   Exemplo:
   - UI: padronizar tipografia da tela clusters
   - Fix: corrigir tooltip no modo escuro
6. Clicar em "Commit to main"
7. Clicar em "Push origin"

Pronto. Projeto atualizado no GitHub.

---

## Se algo der errado
- NÃO usar git pull / rebase / merge manual
- Fechar tudo
- Abrir GitHub Desktop
- Usar o botão "Fetch / Pull"

---

## Regra de ouro
- Antigravity = editar código
- GitHub Desktop = salvar versão oficial
- ChatGPT = decidir o que mudar e revisar
